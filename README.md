# MyDiary

### Тестовое задания от [ADWIN.agency](https://adwin.agency/)

## Описание проекта

Личный дневник.

## Функционал:

- Добавление записи
- Пагинация записей
- Сортировка записей

## Стек технологий:

### Backend:

- TypeScript
- NestJS
- MongoDB

### Frontend:

- TypeScript
- React
- Redux Toolkit
- RTK Query

## Установка и запуск проекта:

Клонировать репозиторий:

    git clone https://github.com/Kirill-Karpachev/my-diary.git

### Если есть Docker:

    docker compose up

### Если нет Docker:

Перейти в папку backend:

    cd backend

Установить зависимости:

    npm install

Запустить проект:

    npm run start

Перейти в папку frontend:

    cd frontend

Установить зависимости:

    npm install

Запустить проект:

    npm run dev
